# Wedding Invitation (Hwan-young & Ji-won)

This repository contains the source code for Hwan-young and Ji-won's mobile wedding invitation.

## Features

- **Responsive Design**: Works on mobile and desktop.
- **Event Details**: Date, Time, and Location.
- **Location**: Integration with Kakao Map and Naver Map for easy navigation.
- **Gallery**: Photo gallery of the couple.
- **RSVP**: Link to a Google Form for guest responses.
- **Countdown**: A countdown timer to the wedding day.

## Setup

The project consists of static HTML, CSS, and JavaScript files. No build process is required.

To view the site locally:
1. Open `index.html` in your web browser.

## Deployment

The site is designed to be hosted on GitHub Pages or any static site hosting service.

- **Main File**: `index.html`
- **Styles**: `css/style.css`
- **Scripts**: `js/script.js`

## Customization

- **RSVP Link**: Update the `href` in the RSVP section of `index.html` with your Google Form URL.
- **Map Links**: Update the Kakao Map and Naver Map links in `index.html` with the correct coordinates/place ID.
